# 📌 참여 방법

## Organization 안에서 Repository 생성

GitHub 상단에서:
- **New repository** 클릭
- **이 템플릿 선택 (Use this template)**

1. 공부 내용 작성 또는 코드 수정
2. 변경사항 커밋
3. main 브랜치에 push
